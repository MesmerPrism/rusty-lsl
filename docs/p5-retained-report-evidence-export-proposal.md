# P5 Retained Report Evidence Export Proposal

This frozen P52 lane adds one substantive crate-private Morphospace advisory export proposal over actual P50 retained comparative reports and actual P51 report histories and evidence pages. The caller supplies one complete history and a caller-ordered collection of complete pages. A successful proposal retains those owners unchanged, including every nested proposal, snapshot, report, sample, vector, and allocation. Its separate manifest copies only the existing `Copy` evidence facts.

Construction has explicit nonzero report, page, and combined-evidence bounds. It checks every collection addition, converts every count and index fallibly to `u64`, and reserves the exact manifest length with `try_reserve_exact`. All counts, bounds, arithmetic, conversions, and allocation are preflighted before the first manifest entry is written. Report-limit, page-limit, combined-evidence-limit, overflow, conversion, and allocation failures return the complete original history and page collection with no externally visible partial proposal.

Manifest order is deterministic: all history reports in retained caller order and each report's evidence order, followed by all pages in caller order and each page's retained evidence order. Entries identify their exact report or page position, evidence position, and global export position; page entries also carry the page's existing start, end, and total positions. Zero evidence is representable when the supplied retained owners contain none, exact bounds succeed, and one-past bounds fail closed.

The proposal is caller-requested, default-inert, advisory, and non-applying. It is not serialized as or exposed through a public protocol, does not infer missing evidence, loss, continuity, or causality, and claims no liblsl equivalence. It has no root/runtime export or application path and acquires no Manifold, session, stream, transport, control, activation, admission, routing, device, oracle, recovery, discovery, or policy authority.
